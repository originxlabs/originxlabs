## Local Development

Requirements:
- Node.js 18+
- npm

Install and run:

```sh
npm install
npm run dev
```

Production build:

```sh
npm run build
npm run preview
```

## Tech Stack

- Vite
- TypeScript
- React
- Tailwind CSS
